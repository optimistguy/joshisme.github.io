My FREE Data Science and Machine Learning Master’s Degree!

Hi everyone! This is my first blog post sooooooooooo yay. The reason why I want to write this is because I want to learn Data Science and Machine Learning and have experienced with some part of it! Ok so let get started!

The basics, everyone need to start somewhere:
  1. Harvard’s CS50 Course (Intro to computer science.)
  2. Intro to programming (A cut-down version of the Nanodegree. But for me that is enough cause you also doing the CS50 course!)
Now for some algorithms!, important!
  1. Algorithms on KhanAcademy (great course!)
  2. Data Structures & Algorithms in Python (for some data scutures)

The basics will take about 1 - 2 months

Now for some maths :
  1. Linear algebra - Khan Academy
  2. Calculus - Khan Academy
  3. Multivariable calculus - Khan Academy
  4. Statistics and probability - Khan Academy
  5. Vectors - Khan Academy
  6. Matrices

That is all for maths. There are technologies that remove the math part but you need to learn math to know the core. Technologies change, math stay the same!

Ok, so you have done math and are the basics of computer science. Now we will have the basics of Data Science and Machine Learning!

Data Science Track:
  1. Introduction to Data Science
  2. Intro to Inferential Statistics (you have learn statistics before, now for some inferential statistics)
  3. Intro to Descriptive Statistics (for more statistics!)
  4. Intro to Data Analysis
  5. SQ: for Data Analysis
  6. Data Wrangling with MongoDB
  7. Data Visualization and D3.js
Machine Learning:
  1. Intro to Machine Learning
  2. Deep Learning
  3. fast.ai

I will keep this up to date so make sure to go back and check, there will be lots of still coming. This bot the complete version yet cause I am on vacation right now.
